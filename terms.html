MinaTech Realty Console 利用規約

最終更新日: 2026年4月12日 ／ 施行日: 2026年4月12日

本利用規約（以下「本規約」）は、MinaTech株式会社（以下「当社」）が提供する「MinaTech Realty Console」（以下「本サービス」）の利用条件を定めるものです。本サービスをご利用いただく前に、本規約を必ずお読みください。本サービスの利用をもって、本規約に同意したものとみなします。

第1条（定義）

  1. 「本サービス」とは、当社が提供するMinaTech Realty Consoleウェブアプリケーション及び関連する一切のサービスをいいます。
  2. 「利用者」とは、本サービスを利用する個人又は法人をいいます。
  3. 「ライセンスキー」とは、当社が利用者に対して発行する本サービスの利用許諾を示す認証コードをいいます。
  4. 「レインズ」とは、公益財団法人不動産流通推進センターが運営する不動産流通標準情報システム（Real Estate Information Network System）をいいます。

第2条（サービスの概要）

  1. 本サービスは、利用者がレインズ等から手動で取得（コピー）した物件情報テキストを、ブラウザ上でローカルに解析・評価する支援ツールです。
  2. 本サービスは、レインズへの自動アクセス、スクレイピング、クローリング等の自動取得行為を一切行いません。
  3. 本サービスにおける全ての処理は、利用者のブラウザ内で完結し、物件データを外部サーバーに送信することはありません。

第3条（利用資格）

  1. 本サービスは、宅地建物取引業法に基づく免許を有する宅地建物取引業者（以下「宅建業者」）及びその従業員が利用することを前提としています。
  2. 利用者は、レインズの利用規約、利用規程その他レインズが定める一切のルールを遵守するものとします。
  3. 利用者は、宅建業免許番号その他当社が求める情報を正確に提供するものとします。

第4条（ライセンス）

  1. 本サービスの利用には、当社が発行する有効なライセンスキーが必要です。
  2. ライセンスの種類は以下のとおりとします。
プラン 利用期間 利用人数 機能
トライアル 14日間 1名 基本解析・CSV出力
スタンダード 1年間 3名まで 全機能（Excel/JSON含む）
プロフェッショナル 1年間 10名まで 全機能＋評価基準カスタマイズ
  1. ライセンスキーは、発行された利用者（法人の場合はその組織）のみが使用できるものとし、第三者への譲渡、貸与、共有は禁止します。
  2. ライセンスの有効期間が満了した場合、利用者は本サービスの有償機能を利用できなくなります。

第5条（禁止事項）

利用者は、以下の行為を行ってはなりません。

  1. 本サービスを利用して、レインズの利用規約に違反する行為
  2. レインズから取得した情報を、宅建業法及びレインズ規程に反して第三者に提供・公開する行為
  3. 本サービスのソースコード、アルゴリズム、デザインを複製、リバースエンジニアリング、改変する行為
  4. 本サービスの評価ロジックを用いて、競合サービスを構築する行為
  5. ライセンスキーを不正に生成、複製、又は第三者に開示する行為
  6. 虚偽の宅建業免許番号を用いて利用登録を行う行為
  7. 本サービスを利用して自動化ツール（ボット、スクリプト等）を構築する行為
  8. その他、当社が不適切と判断する行為

第6条（知的財産権）

  1. 本サービスに関する著作権、特許権、商標権その他一切の知的財産権は、当社に帰属します。
  2. 本サービスの物件評価アルゴリズム（10点満点スコアリングシステム）は、当社が独自に開発したものであり、当社の営業秘密として保護されます。
  3. 利用者が本サービスを通じて出力したデータ（CSV、Excel、JSON等）の著作権は利用者に帰属します。ただし、出力データに含まれる評価ロジックの仕組み自体は当社に帰属します。

第6.5条（重要事項説明書下書きジェネレータの位置づけ）

  1. 本サービスが提供する「重要事項説明書 下書きジェネレータ」は、宅地建物取引業法第35条に基づく重要事項説明書を、物件マスタDBの情報から下書きとして自動生成する支援機能です。
  2. 本機能で生成された下書きは、それ自体では宅地建物取引業法上の効力を持ちません。実際の重要事項説明は、宅地建物取引士による対面又はIT重要事項説明（IT重説）を通じて、説明を行い、当該書面に記名押印（書面）又は電子署名（IT重説）した上で初めて法的効力を持ちます。
  3. 下書きに記載される事実情報（登記事項、法令制限、私道負担、インフラ状況、ハザード等）の真正性・完全性は、利用者である宅地建物取引業者及び宅地建物取引士の最終確認義務に属します。当社は下書きの内容に起因する取引上のトラブル、行政処分、損害について一切の責任を負いません。
  4. IT重説対応機能は国土交通省「ITを活用した重要事項説明に係るマニュアル」遵守の補助情報を生成しますが、通信状態確認、本人確認、書面到達確認等の重説実施手続きの完遂義務は宅地建物取引士本人にあります。
  5. 下書き機能は MinaTech Realty Console Standard プラン以上で利用可能とします。

第7条（免責事項）

  1. 本サービスが提供する物件評価は、当社独自のアルゴリズムに基づく参考情報であり、投資判断の最終決定は利用者自身の責任で行うものとします。
  2. 当社は、本サービスの評価結果の正確性、完全性、最新性について保証するものではありません。
  3. 当社は、本サービスの利用に起因して利用者に生じた損害について、当社に故意又は重大な過失がある場合を除き、一切の責任を負いません。
  4. 当社の損害賠償責任は、いかなる場合も、利用者が当社に支払ったライセンス料の直近1年分を上限とします。

第8条（レインズ規約の遵守）

重要: 本サービスは、レインズのデータを自動取得する機能を有しません。利用者がレインズから物件情報を取得する際は、必ずレインズの利用規約及び利用規程を遵守してください。レインズの利用規約に違反して取得されたデータの利用について、当社は一切の責任を負いません。

  1. 利用者は、レインズのデータを手動でコピーする際、レインズが定める利用目的の範囲内で行うものとします。
  2. 利用者は、本サービスで解析したデータをレインズの利用規程に反して利用しないものとします。
  3. レインズの利用規約変更等により本サービスの運用に影響が生じた場合、当社は本サービスの内容を変更又は停止できるものとします。

第9条（個人情報の取り扱い）

  1. 本サービスは、利用者がテキストエリアに入力した物件データを外部サーバーに送信しません。全ての解析処理はブラウザ内で完結します。
  2. ライセンス認証に必要な情報（会社名、宅建業免許番号、メールアドレス等）は、当社が別途定めるプライバシーポリシーに従い管理します。

第10条（サービスの変更・停止）

  1. 当社は、利用者への事前通知をもって、本サービスの内容を変更又は停止できるものとします。
  2. 緊急の場合、当社は事前通知なく本サービスを停止できるものとし、この場合は速やかに利用者に通知するものとします。

第11条（料金・支払い）

  1. 本サービスの料金は、当社が別途定める 料金プラン によります。料金は新規契約者に即時適用され、既存契約者には次回更新時より適用されます。
  2. 年契約プランの料金は契約時に全額前払い、月払いプランの料金は前月までの銀行振込前払いとします。
  3. 支払いが滞った場合、滞納2週間で自動的に利用を停止し、滞納2ヶ月でアカウント・データを削除します。入金確認後の再開時、3,000円の再開手数料を申し受けることがあります。
  4. 年契約期間中の途中解約および返金は受け付けておりません。月払いプランの場合、当月末日まで利用可能とし、翌月分から自動停止します。

第12条（サポート範囲）

  1. 本サービスのサポートはメールに限ります。電話・チャット等の即時対応は行いません。
  2. 各プランのサポート時間上限：
    • Trial: サポート対象外（自動失効）
    • Standard: メール月1時間まで
    • Professional: メール月3時間まで・優先対応・初回データ移行支援1回
  3. サポート時間は1往復あたり15分換算でカウントします。仕様確認・調査は実所要時間です。緊急バグ修正はサポート時間に含めず、プランを問わず最優先で対応します。
  4. 月のサポート時間上限到達時、翌月への繰越は行いません。超過分の対応は別途有償となります。
  5. 機能要望は Standard 以下では Roadmap への登録のみで、実装時期は保証しません。Professional プランの利用者は、年2件までカスタマイズ枠を利用できます。

第13条（契約解除）

  1. 利用者が本規約に違反した場合、当社は催告なくライセンスを取り消し、本サービスの利用を停止させることができます。
  2. 前項の場合、当社はライセンス料の返金義務を負いません。

第14条（準拠法・管轄）

  1. 本規約は、日本法に準拠し、日本法に従って解釈されるものとします。
  2. 本規約に関する紛争については、横浜地方裁判所を第一審の専属的合意管轄裁判所とします。

第15条（規約の変更）

  1. 当社は、必要に応じて本規約を変更できるものとします。
  2. 変更後の規約は、当社が別途定める場合を除き、本サービス上に掲載した時点で効力を生じるものとします。
