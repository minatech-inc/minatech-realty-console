# MinaTech Realty Console チャットボット用ナレッジ

このドキュメントは AIチャットボットが参照するナレッジソース。System Prompt として Cloudflare Workers からClaude API に送信される。

## 主要機能

1. **物件評価スコアリング**: レインズコピペで10点満点の自動評価。S/A/B/Cランク判定（S=7点以上即日対応、A=5-6点今週中、B=3-4点今月中、C=2点以下見送り）
2. **SUUMO入稿支援**: SUUMO入稿規定2025.12版準拠、特徴項目180種自動判定
3. **画像処理**: SUUMO規格3プロファイル自動リサイズ、face-api.jsで顔自動マスキング、Tesseractでナンバープレート検知、4MB圧縮
4. **物件マスタDB**: IndexedDB永続化、9ステータス管理（検討中/注目/問合せ済/内見予定/内見済/指値中/契約済/失注/保管）、タグ・メモ
5. **ポータル横断チェッカー**: 物件名で SUUMO/atホーム/HOMES の該当ページを即特定。住所からも市区町村単位ジャンプ可能
6. **銀行担保評価書**: 積算×80% / 収益還元×90% の低位採用でPDF自動生成
7. **国交省API統合**: 不動産情報ライブラリ 28空間API（取引価格・地価公示・ハザード11種・都市計画7種・生活環境10種・将来人口）

## スコアリング基準（10点満点）

- エリア市場性: 0〜+3点（東京23区/政令市=3点、都下/TIER1主要県=2点、TIER2地方中核=1点、TIER3郊外=0点、過疎地=減点）
- 表面利回り: 0〜+2点（15%以上=2点、10%以上=1点）
- 駅距離: 0〜+1点（10分以内=1点）
- 価格帯: 0〜+1点（500万円以下=1点）
- 賃貸中(OC): 0〜+1点（入居中=1点）
- 構造・築年: 0〜+1点（RC造または築浅=1点）
- 再建築不可: -3点
- 実質利回り: 0 or -2点（実質5%未満=-2点）

## 業界課題カバー範囲（LPで開示済み）
