# MinaTech Realty Console 価格・運用ポリシー

**社内ドキュメント** / 制定 2026-05-18 / 改定責任者: 磯谷 肇（MinaTech株式会社 代表取締役）

このドキュメントは「ASKって何円ですか？」と問われた時に、即答できるように指針を文書化したもの。
営業・サポート対応時の判断基準として参照すること。

---

## 1. 設計原則

**MinaTech 株式会社のリスク（時間・追加コスト・法的）を最小化する**ことを最優先とする。
「機能で勝負」ではなく「明文化と前払いで守る」を基本方針とする。

### リスク → 防御策

| リスク要素 | 防御策 |
|---|---|
| サポート時間負荷 | プラン別にサポート時間上限を明記、電話なし・メールのみ |
| 価格交渉時間 | 公開価格・Web申込のみ・個別見積なし（PROカスタマイズ除く） |
| カスタマイズ要望 | カスタマイズは PRO のみ・年2件まで。追加は別途見積 |
| 債権回収 | 年契約は前払い、月払いは銀行振込前払い、滞納2週間で自動停止 |
| 解約処理 | 年契約=途中解約不可、日割り還付なし |
| 投資判断トラブル | terms.html で投資助言業者でない旨明記 |
| データ消失 | JSONバックアップ提供で「データ管理はお客様責任」と明示 |
| 個人情報漏洩 | 画像処理ローカル完結を技術的に担保 |

### 2.2 業界実勢との比較（根拠）

| 業界実勢 | 月額相場 | Realty Console との比較 |
|---|---|---|
| SUUMO公式 業者向け入稿ツール | 3〜5万円/月 | Standard はその1/5〜1/8 |
| いい生活CRM / HOME'Sプロ等 | 数万円/月 | 同等以下で複数機能カバー |

→ **複数機能統合**であることが差別化ポイント。営業時に必ず強調する。

### 2.3 算定根拠（時間価値ベース）

宅建営業の時給 3,000〜5,000円。1物件あたり Realty Console が短縮する時間は約3時間（=9,000〜15,000円の価値削減）。
**月10物件処理する業者なら、Standard 5,980円/月 は1物件分の節約で元が取れる**水準。

## 3. 解約・トラブル時の標準条項

### 3.1 解約規定

- **年契約途中解約**: 不可、残期間返金なし
- **月払い解約**: 当月末まで利用可、翌月分から停止（次月決済を止めるだけ）
- **トライアル期間中**: いつでも自動失効、課金一切なし
- **無断解約・連絡なしの停止**: ライセンスキーを自動無効化

### 3.2 未払い対応

- **滞納2週間**: 自動利用停止（データは保管）
- **滞納2ヶ月**: アカウント削除、データ消去
- **入金確認後**: 即時再開（手数料 3,000円申し受けることあり）

### 3.3 機能要望対応

- **Trial / Standard**: Roadmap への投票のみ（実装時期未保証）
- **Professional**: 年2件まで優先実装枠（仕様調整は別途協議）
- **緊急バグ**: プラン問わず最優先で修正

### 3.4 データ移行サポート

- **Trial / Standard**: 自己責任（JSONバックアップ機能利用）
- **Professional**: 初回移行を1回まで MinaTech 側で支援

### 3.5 サポート時間カウントルール

- メール返信: 1往復 = 15分換算
- 仕様確認・調査: 実所要時間
- バグ修正: サポート時間に含めない
- 月のサポート時間上限到達時: 翌月繰越なし、超過分は応相談（PROは延長可、Standardは追加見積）

## 4. 営業時の即答テンプレート

### 4.1 「料金はいくらですか？」

> Standard が **年49,800円（月払い5,980円）** で3名まで、Professional が **年148,000円（月払い14,800円）** で10名まで、です。
> 14日間の無料トライアルでまず試していただけます。

### 4.2 「他社より高い/安い理由は？」

> 物件評価・SUUMO入稿・画像処理・物件マスタ・銀行担保評価書を**一画面で全て**まかなえる点が違います。
> 他社の単体ツールを組み合わせるより**1/3〜1/5の価格帯**になっています。

### 4.3 「途中解約はできますか？」

> 年契約は途中解約と返金対応はしておりません。**月払いプラン**であれば、当月末で停止できます。
> ご不安な場合はまず月払いから始めて、慣れてから年契約に切り替えることも可能です。

### 4.4 「カスタマイズしてほしい機能があります」

> Professional プランの「年2件まで優先実装枠」でご対応可能です。
> Standard プランの場合は Roadmap への要望登録（投票制）となりますが、緊急バグ以外の実装時期はお約束できません。
> どうしても優先実装が必要であれば、別途お見積もりもご相談承ります。

### 4.5 「データはどこに保存されますか？／消えませんか？」

> お客様のブラウザ内（IndexedDB）に保存され、クラウドへは送信しません。
> 端末移行や引継ぎのために **JSONバックアップ / 復元機能** を提供しています。
> Standard 以下は自己責任、Professional は初回移行を1回まで弊社で支援します。

### 4.6 「スコアの根拠は？」

> 10点満点の採点基準・重み配分・判定基準を `https://realty.minatech1210.com/scoring.html` で **完全公開** しています。
> ブラックボックスのAIではなく、根拠を明示したルールベースのエンジンです。
> ただし投資判断の最終決定はご利用者ご自身の責任となります（金融商品取引法上の投資助言業者ではありません）。

## 5. 価格改定時の方針

- 改定は**新規契約者**にのみ即時適用、既存契約者は次回更新時から
- 改定告知は最低3ヶ月前にメール通知
- 値下げ時は既存契約者にも遡及適用なし（次回更新時のみ）
- Trial の0円・14日設定は維持（CAC 削減のため変更しない）
